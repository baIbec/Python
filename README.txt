Описание
Простое приложение на Python с графическим интерфейсом Tkinter для генерации, хранения и фильтрации цитат известных авторов. Позволяет создавать случайные цитаты, сохранять их в файл, удалять ненужные и быстро находить нужные по автору или теме.

Примеры использования
- Генерация цитаты: нажмите кнопку «Генерировать цитату» — новая случайная цитата появится в списке и сохранится в файл.
- Фильтрация: введите имя автора (например, «Толстой» или тему (например, «Мудрость») в соответствующие поля и нажмите «Фильтровать». В списке отобразятся только подходящие цитаты.
- Сброс фильтра: чтобы вернуть полный список цитат, нажмите «Сбросить фильтр».
- Удаление: выделите цитату в списке и нажмите «Удалить выбранную» для её удаления из списка и файла.
